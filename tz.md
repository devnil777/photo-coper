Цель: терминальное приложение, с TUI выполняющие задачу копирования изображений с флешки камеры на жесткий диск

Целевая платформа: windows
Скрипт на python

Требования:
- просканировать все внешние диски, найти все флешки
- в один момент времени может быть подключено несколько флешек
- файлы которые нас интересуют: *.cr2, *.cr3, *.raf (маски задаются в конфиг файле)
- в настройках может быть задано несколько каталогов куда копировать файлы (например, c:\photo)
- файлы должны быть сгруппированы по датам, в общем случае один день - одна группа
- пользователь может выбрать группы (checkbox)

алгоритм работы:
утилита сканирует все диски (съемные диски) ищет на них изображения

указывает пользователю в формате


Шаг 1
список:
дата (объем) кол-во изображений, где были найдены, расширения
считаем что файлы должны быть в папке DCIM 

так же выводи предупреждение если в множестве файлов есть пересечение по именам (если на камере сбилась нумерация)

пользователь может выбрать несколько папок (групп по датам), есть пункт обновить список

если список содержит только одну запись просто выводим эту информацию и переходим к следующему шагу

Шаг 2
пользователь выбирает (из списка) каталог куда он хочет копировать эти файлы

если выбранном списке только одна дата, то пользователю предлагаю задать имя
имя папки в итоге строиться как дата + имя: YYYY-MM-DD <название>

если несколько дат, то пользователь вводит имя всей папки целиком

Шаг 3
после ввода имени
осуществляется проверка, есть ли указанная папка и есть ли в ней файлы

если есть, то пользователя простят подтвердить или вернуться на предыдущий шаг

Шаг 4
если среди выбранных папок есть пересечения по именам, то у пользователя просят подтвердить что файлы будут скопированны в под папки типа 1\ , 2\ , 3\ и т.д.

Шаг 5
вопрос пользователю копировать или удалить файлы после копирования из исходных каталогов 

Шаг 6
вопросы:
нужно ли после копирования создать каталог LR
запустить ли Lightroom

Шаг 7
перед начало выполнения операции, выводиться весь выбор какой сделал пользователь что бы его подтвердить

копирование осуществляется по очереди из каждой выбранной папки (гурппы) в каталог назначения, ы папку src 
если надо файлы помещаются в разные папки 1\ , 2\ , 3\

Выводиться общий прогресс 

Файл должен быть скопирован максимально надежным способом, так же у файла должно быть перенесено дата и время 

Если файл есть и он идентичный (по размеру и времени) -- пропускаем, во всех остальных случая файл копируется повторно

Если ошибка чтения копирование прекращается, выводит ошибку и ждет подтверждения от пользователя

в настройках задается путь откуда скопировать структуру файлов (рекурсивно) базовые файлы для каталога Lightroom, если файлы уже есть пропускаем копирование

после успешного копирования выводит информацию сколько было скопировано, сколько это заняло времени

если надо удаляет исходные файлы

запускает lightroom если надо 

и ждет подтверждения от пользователя для завершения

прогресс копирования должен выводиться в windows taskbar 

запускать приложение буду в windows terminal

приложение на Python

скрипт сборке exe-файла

конфиг файл находится в домашнем каталоге пользователя

TUI интерфейс:
это списки или строки ввода, между которыми можно перемещаться стрелками, полноценный ввод
на каждом шаге можно или вернуться на предыдущий шаг, или завершить приложение
шаги по очереди выводятся на консоль, и результат предыдущего шага остается на консоле


приложение не завершается, без подтверждения от пользователя
