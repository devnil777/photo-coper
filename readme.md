# Photo Coper

Терминальное приложение (TUI) для автоматизации копирования фотографий с камер (флешек) на жесткий диск.

## Возможности
- Автоматический поиск съемных дисков и папок `DCIM`.
- Группировка файлов по датам.
- Обнаружение конфликтов имен (когда камера сбрасывает счетчик).
- Разрешение конфликтов через создание подпапок (по датам или номерам).
- Копирование с сохранением времени изменения файла.
- Пропуск уже скопированных идентичных файлов.
- Удаление исходных файлов после успешного копирования (опционально).
- Создание структуры проекта Lightroom из шаблона.
- Прогресс-бар в консоли.

## Установка

1. Установите Python 3.10+
2. Установите зависимости:
   ```bash
   pip install questionary pyyaml psutil tqdm pywin32
   ```

## Конфигурация

Файл настроек `.photo-coper.yaml` создается в домашнем каталоге пользователя при первом запуске.

Пример содержимого:
```yaml
destination_directories:
- C:\Photos
- D:\Archive
extensions:
- '*.cr2'
- '*.cr3'
- '*.raf'
- '*.jpg'
lightroom_exe: C:\Program Files\Adobe\Adobe Lightroom Classic\Lightroom.exe
lr_template_path: C:\Templates\LR_Standard_Project
```

## Использование

Запустите скрипт:
```bash
python -m photo_coper.main
```

Или используйте собранный EXE файл.

## Сборка EXE

Для сборки в Windows используйте `build.bat`. Вам потребуется установленный `pyinstaller`.
```bash
build.bat
```
Результат будет в папке `dist/photo-coper.exe`.
